# SmallAI Projects

A collection of small-scale, responsible, and inclusive AI projects. Explore projects by value clusters, categories, and tags.  

# What is SmallAI
Smallness is not simply a technical descriptor (model size or computational resources), but a deliberate value-driven stance. Projects framed small AI around transparency, efficiency, data sovereignty, feminist ethics, ecological sustainability, and community-driven approaches, indicating a broad cultural shift toward more accountable AI.
